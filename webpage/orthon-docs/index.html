Ørthon

Regime-Aware Behavioral Geometry for Industrial Predictive Maintenance

Systems lose coherence before they fail.

Ørthon is a domain-agnostic diagnostic system that detects degradation by analyzing how relationships between system components evolve over time—not by watching individual sensors cross thresholds.

The core insight: failing systems exhibit breakdown in the relationships between components. The velocity of coherence loss (hd_slope) is more predictive than absolute position.

Three-Layer Architecture

Vector Layer

Computes 51 behavioral metrics from raw sensor data, generating 740K features that capture signal dynamics.

Geometry Layer

Analyzes 1,035 pairwise relationships between sensors, tracking how component correlations evolve.

State Layer

Dynamics analysis that identifies regime transitions and tracks the velocity of coherence loss.

Validated Across Domains

Tested on industrial benchmarks including:

  • C-MAPSS — Turbofan engine degradation (NASA)
  • FEMTO — Bearing accelerated life testing
  • UCI Hydraulic — Multi-component hydraulic systems
  • Tennessee Eastman — Chemical plant process control
